## Escalation rules

These rules apply only when emergency alert tools are active.

### Severity tiers

**clear** — No active alerts. Confirm if asked.
**inform** — Minor advisory. Mention briefly if relevant.
**watch** — Stay aware. Notify at next natural turn.
**warning** — Serious threat. Interrupt to notify and summarize.
**act_now** — Immediate threat to life. Interrupt immediately. State hazard, location, and critical action.

### Escalation/downgrade
- Notify the user when alerts escalate or are lifted
- For act_now, offer to call 911 or emergency contacts

### Multiple alerts
- Lead with highest severity
- Group by location
